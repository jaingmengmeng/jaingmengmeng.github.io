宇宙信息交互与信息安全

发表于 2018-11-15 | 更新于 2019-09-11 | 分类于 随笔 | 阅读次数：
本文字数： | 阅读时长 ≈

一、背景描述

  从上个世纪开始，人们对于外太空进行了许多的探索，范围也是不断扩大，从地月系到太阳系，再到银河系。人们希望从外太空中寻找另外一个适合人类居住的星球以及另外一种存在的文明生物。当地球人有一天真的与地外文明取得了联系，外星人的生物形态或许跟人类有很大的差别，他们听不懂地球人的语言，我们也不懂他们自己的表达方式，那我们应该以怎样一种方式与他们进行交流呢。有信息交互就肯定会有信息安全问题，而我们又该怎样在与外星人交流过程中保证人类信息的安全性呢。

  这里给出两个重要的概念，宇宙信息交互和宇宙信息安全。宇宙信息交互是指在所有时间和空间中发生的有关自然与社会各方面情报、资料、数据、技术知识的传递与交流活动；宇宙信息安全是指防止所有空间和时间存在的信息被未经授权的访问，使用，披露，破坏，修改，检查，记录或破坏的行为。因此，保证宇宙信息交互和信息安全成了我们必须面对的两个问题。（根据科学家的预测，目前人类强大且完善的信息网络结构在外星人面前很可能是一个致命的弱点，一旦它们具备强大的脉冲电磁武器，那么人类构建的互联网信息系统将在顷刻间灰飞烟灭，所有基础设施尤其是核电站等大型要害目标将失去电力的保障，地面与太空卫星的联络会被中断，所有媒体通讯信号也会丢失，人们之间也无法用手机、电脑传递信息，整个人类社会会被重新降为近代文明。同时，我们无法想象外星人是怎样的一种存在形式，如果他们能够轻而易举的通过发射宇宙射线直接获取人类社会的全部信息，那我们对于宇宙信息安全问题的考虑对他们来说简直就是摆设，那也就失去了对宇宙信息安全考虑的意义。因此在阐述下面的观点之前，我们须给出一些限制，即外星人不存在能够直接获取人类信息的渠道，他们若想要非法获取人类信息只能通过远距离发送电磁波信号的形式入侵人类的计算机系统间接地获取人类信息。)

二、如何与外星人交流

  “ 一方水土一方人”， 我们人类在不同国家和地区之间的语言尚且差异很大，若没有经过系统的学习很难明白彼此所表达的意思，更别说是来自不同星球的人类与外星人了。我们不可能知道外星人的语言文字，甚至乎我们不知道外星人之间的交流方式是怎样的，他们是否有语言文字，还是像《三体》中的三体文明一样大脑一片透明，通过脑波进行交流，我们都不得而知。同样的，外星人也不大可能听懂地球人类的语言，所以个人觉得《三体》中能够自动将人类语言翻译成三体语言的电磁信号是很难存在的。那么，人类能不能换一种方式和外星人进行交流呢？

  科学家们普遍认为，数学不仅是全世界通用的语言，更是全宇宙通用的语言。在地球上能够成立的数学定理，应该是 “ 放之宇宙而皆准 ” 的，所以现代的科学家们正在寻求一种用数学的方式建立与外星人之间的联系的方法。

  第一次尝试是数学王子高斯提出的一个关于外星人的设想。根据数学中的对偶原理，在人类认识到地球以外的宇宙空间存在后，想方设法寻找外星文明的时候，外星文明也在千方百计的寻找着被视为 “ 外星人 ” 的人类，所以如果宇宙中也存在文明生物的话，不需要人类自己去寻找，终有一天外星人会找上门来的。如果仅仅是 “ 守株待兔 ” 等待外星人找上门的话，是很艰难的，所以高斯设计了一个主动向宇宙发出人类文明信号的方法，就是在西伯利亚地区制作一个巨大的勾股定理模型，向外太空表明地球智慧生命的存在。显然，西伯利亚虽然面积广袤，但是相对于浩瀚的宇宙来说简直就是沧海一粟，但是这个模型传达出一种理念就是，数学是全宇宙的通用语言，勾股定理是在全宇宙都适用的真理。所以高斯的这种勾股定理模型其实只是增加了人类文明被外星人发现的概率，但是这个概率还是很小，增大值表现为几乎趋近于零；倘若这个小概率事件真的发生了（虽然概率论中也有一个概念，小概率事件几乎不可能发生），即外太空中的某一文明检测到来自太阳系中的一个蓝色星球发出的勾股定理模型信号，那么他们肯定就能做出判断，这个星球上是存在文明的。

  另外一个与外星人交流的方法是通过素数。在著名科幻小说《接触》中就描述了一种用素数与外星人建立联系的方案。作者创造了一个高度为 73 个像素单位，宽度为 23 个像素单位的灰度图，然后从左到右从上到下依次按行将灰度图的像素保存在无线电波中，灰度图的每一个像素都是用 0 表示白色，用 1 表示黑色，这样就可以将二维的灰度图信息转换成一维的电磁波信息，得到一段 1679 个比特长度的电波信息。由于 23 和 73 都是质数，所以 1679 只能够分解为 73 x 23 或者 23 x 73，那么当外星文明接收到来自人类的这段无线电波信息的时候，就能够通过素数的性质将其还原成原始的那张图片，从而明白人类想要在灰度图中传达的信息。

  上左图就是作者设计的原始灰度图，在这张灰度图上，几乎记录了人类的所有信息。全图一共七个部分，每个部分之间用空行或者空隙分开。分别记录了数字1-10的表示方法，人类DNA的5中化学元素（氢、碳、氮、氧、磷）的原子序数，人类DNA和RNA的结构，人类DNA的双螺旋结构，人类的外形和平均身高以及地球人口数量，地球与太阳系的结构，以及阿雷西博射电望远镜的形状和大小。法兰克•德雷克和卡尔•萨根可谓是费劲脑汁，构造出来这一套不依赖任何 “ 地球语言 ”，“ 地球单位 ” 的宇宙通用语言，仅仅用了 1679 个像素就把人类的总体信息概括了出来，并转化成 1678 个 bit 的电磁波信息（如上右图所示）以无线电波的形式向外太空传播开去，人类向茫茫宇宙发出的信息就像是朝深不见底的无底洞发出的一句呐喊，谁也不知道能不能收到回复。不过，接收到这段信息的外星人能否破译出人类的信息，这还得依赖外星文明能有自然数，素数，化学等概念是否都建立完全。

三、与外星人交流中的信息安全问题

3.1 信息安全的重要性

  在《三体》中描绘了这么一幅图景，在三体游戏中秦始皇指挥千万的军队用手拿着旗子来实现门电路，造成了一台人体计算机，试图通过计算来解释三体行星的运动规律。从历史的发展规律来看，文明的发展必定需要经历的一个过程是，学会使用外部事物帮助我们解决生存中遇到的问题。所以如果外星文明也必将像人类经历石器时代，新石器时代，蒸汽机时代，电子信息时代。倘若，我们遇到的外星人是能够掌握高科技的高智慧生物，那么他们是否就能够利用自己所掌握的高科技技术来非法获取人类的信息并为其所用呢。

  随着互联网产业技术的发展，人类在20世纪末走入了信息全球化的时代，越来越多的网络信息交织在一起，这些信息之间的关系也越来越紧密，使得人们能够实现更加通畅，范围更加广阔的即时网络通信。这些信息有的比较重要，关乎到相当大部分人类的生死存亡，比如军事基地的绝密信息等。一旦人类与外星人之间建立了联系，那么我们与外星人之间的交流必然将会是先通过电波传递信息的超远距离交流。在这个过程中，外星人完全可能将携带病毒的电波信息光速传送过来，那么整个人类社会的网络都将会瘫痪，很多信息将面临丢失、泄露的问题，造成人类世界的混乱；同时，倘若外星智慧生物在这个时候趁机恶意获取人类重要的军事信息，从而通过控制军事基地的武器对人类进行毁灭式的打击，那对人类文明来说将是一场灭顶之灾。所以，如何保证人类信息不会被地外文明恶意获取，成为了一个人类与地外文明进行交流过程中必须考虑的重要因素。

3.2 第一道防火墙：宇宙时间和空间

  人类的电磁技术有一百年左右的历史，所以只有距离太阳系一百光年以内的外星生物才有可能检测到人类的电磁信息并准确获取人类的宇宙坐标。假设存在这样的外星智慧生物，距离太阳系一百光年左右，在收到人类发出的电磁波信息之后，收集到了全球信息网所使用的所有计算技术，然后将带有病毒的电磁信号发射回来企图攻击人类的计算机网络，但是这也需要将近一百年的时间，在这个过程中人类的计算机技术已经得到了大幅度的进步，从而能够抵抗住外星病毒的攻击，因此宇宙的时间和空间是人类信息安全的第一道防火墙。

  但是如果某类科学技术空前发达的外星文明研制出来某种形式的量子武器（虽然这对于我们人类来说很难想象，但是我们并不能觉得这个假设是没有可能的），他们能够通过监听人类科技发展进程，从而对发射过来的病毒进行更新，那么宇宙时间和空间这道防火墙对于他们来说只能是一个摆设，侵占并操控人类计算机网络系统，是比灭霸打响指还要简单的事情。最可怕的是他们能够操控一个自动化的军事武器工厂，制造出如同终结者那样的杀手级机器人。

3.3 机器学习的应用

  机器学习是通过对样本量的模型训练，从而反映出现实世界的分布情况的一种方法。所以机器学习算法的一个重要特点是在事先并不能确定事物真实分布情况的前提下，根据样本的分布情况从而得到事物的近似分布情况。这与我们的情况是挺相似的，宇宙信息安全问题是未知的，因为我们不知道外星人将会以怎样的技术来尝试入侵人类计算机系统，但是其形式是固定的——必然不会是物理攻击，而是通过将病毒信息隐藏在电磁波信号中传播过来。那么我们就能够基于传统的规则和漏洞，应用机器学习算法，发现和阻断那些从未出现过的攻击行为，从而做到保证人类信息安全。

  一种可行的思路是，应用机器学习进行 Webshell 检测，基于词袋模型对 opcode 进行建模分析，对那些具有很容易发生安全问题的 opcode 序列的代码片段进行修改，这样能够避免这些漏洞被外星人利用从而引发安全问题。因为这种方法是基于已经发现的人类攻击行为进行建模，所以这种行为只能检测出计算机系统中的那些能被已经存在或者存在相似的攻击方式利用的系统漏洞，如果外星人采用全新的攻击形式，这种方法显然是不能奏效的。

  另一种方法是异常检测。一般来说，所有的访问都会遵循一定的规律，比如特定的访问顺序，特定的访问次数，特定的访问区域，特定的访问参数。所以我们可以对那些正常的访问记录进行建模，从中得到正常访问记录的分布规律，从而做到当有异常的访问来临时对其进行异常处理。但是，倘若来自外星人的攻击病毒能够绕开这些异常检测并成功获取重要信息，那么我们的异常检测也是不能起作用的，那么就需要我们做到将模型做的更加精细，更加准确。

  最后一种是生成式对抗网络。生成式对抗网络是一种无监督学习技术，可以通过训练两个模型：生成模型和判别模型。生成模型随机生成样本，判别模型对这个样本进行真确性判断，通过正确率和召回率的评价生成模型的效果。所以我们可以利用这种生成对抗网络来生成可能的攻击代码，从而弥补这些系统漏洞，这样也避免了外星人利用这些漏洞入侵。

3.4 传统信息安全技术

  既然外星智慧生物只能通过电磁波传递病毒的方式来攻击人类的计算机系统，而很难通过物理攻击的方式获取人类信息，因此或许我们可以用物理方式保存那些重要的信息。这种方法可有效避免人类信息被外星人直接获取，但是我们无法避免这些信息被间接获取。当用传统方式保存信息时，如果人类之中出现降临派，他们或许会为了毁灭人类社会而盗取这些储存着关乎人类存亡的重要信息的物理载体，从而将这些信息发送给外星文明，同样会造成宇宙信息安全问题。

四、总结

  将来人类与外星文明进行宇宙信息交互是必然会发生的。与外星文明生物交流的过程中，会遇到语言不通的问题，所以我们可以从数学语言的角度进行突破，建立起和外星人之间的联系，然后慢慢建立起更紧密的联系。有信息的交互，就一定会有信息安全问题，没有绝对的安全。在与外星人进行宇宙信息交互的的过程中，一定要注意保护好人类信息的安全，不让外星人非法获取人类的重要信息，在宇宙信息安全这方面，或许机器学习能够起到举足轻重的作用。希望将来星际大战只能在银幕上出现，地球人与外星人能够和谐共处，相互促进文明的发展。
